Part I. Intro and Working with Spatial Data

9:00 - 10:15 am

  • Introduction

    • Workshop overview, resources, R vs GIS, RStudio

  • Syntax Review

    • Syntax rules, variables, naming styles, data types, sequences, basic plotting, functions, file names and directories, saving your work

  • Working with Packages

    • Finding packages, spatial package ecosystem, tidyverse, installing and loading packages, what’s in a package

  • Importing and Basic Plotting of Vector Data

    • Spatial data classses; importing from Shapefiles, KMLs, geodatabases, and GeoPackages; basic plotting

  • Visualization Part II

    • Overlaying layers, attribute driven symbology, tmap, leaflet

  • Working with Color

    • Color models, color functions, RBrewer

  • Geoprocessing

    • Geoprocessing from sf package, dplyr examples, randomization

  • Spatial Queries

    • Spatial queries, finding nearest neighbors

  • Raster Pt III. Raster Analysis Functions
